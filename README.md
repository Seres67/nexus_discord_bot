# Nexus Discord Bot

## Prerequisites

- Rust (https://rustup.rs/)

## How to run

Create a file named `.env` at the root of this project with this content:
```
DISCORD_TOKEN=put your token here
```
then either run `cargo run --release` to build and run the bot from source or, for Windows, download the latest action build from the [Actions](https://github.com/Seres67/nexus_discord_bot/actions) tab, or for Linux, download the latest release.
